## 开篇

欢迎来到我的博客！这是我使用 Nuxt 4 和 Nuxt Content v3 构建的第一篇文章。

## 关于这个博客

这是一个基于 Nuxt 4 的现代化博客系统，具有以下特性：

::card-list
- **响应式设计**：完美适配移动端和桌面端
- **夜间模式**：支持深色主题，保护眼睛
- **静态生成**：快速加载，SEO 友好
- **丰富的组件**：支持多种 Markdown 扩展和自定义组件
::

## 技术栈

本博客使用了以下技术：

- Nuxt 4 - Vue.js 全栈框架
- Nuxt Content v3 - 内容管理系统
- Vite - 下一代前端构建工具
- TypeScript - 类型安全的 JavaScript
- UnoCSS - 原子化 CSS 引擎
- Cloudflare Pages - 静态网站托管
